# IronLog
Personal workout tracker for the purposes of logging sessions and motivating towards PRs

### User stories
- I want to track my workouts
- I want to be able to see how I did in the past 1-2 sessions of a given plan day
- I want an option to combine certain exercises as a superset
- I want to choose what plan day I am having this session
- I want to track the time it took me to complete a workout
- I want to see a calendar which displays all my workouts and their details
- I want to have an option to skip optional workouts for abs
- I want to be able to choose a variation of exercises for a muscle group in a few cases
- I want to display PR for each exercise

## Features
- Workout log
- Plan day selection
- Timer which counts until workout is finished
- Calendar
- Option to skip abs exercise
- Option to swap certain exercises for another
- List of exercises with PR display

## Architecture
For code organization, MVC architecture is used
